  • Date: October - November 2015

FileStAck

A Distributed File System implemented in Java.

FileStAck was a project in the Distributed Systems class. I had to write the Remote Method Invocation (RMI), instead of using the Java RMI library. That was used in establishing the communcation between the clients, naming server, and storage servers. I also had to implement the replication and synchronization of files after new writes, for each client to read the lastest version of all files. FileStAck performs the basic functionality of a file system. Clients can read, write, create and delete files, with complete transperancy to the fact that the all data/files is distributed over multiple storage servers.

  • Date: September 2015

Wifi controlled R/C Car

R/C car controlled over wifi network from laptop/computer. Implemented in C.

This was my final project in the Embedded Systems class. Programmed a microprocessor using the FreeScale microcontroller (shown above). It sends pulse signals of various widths to the servos controlling the car, depending on the command sent through wifi module (Xbee) connected to the computer and received by another wifi module connected to the mircroprocessor. The width of the pulse dictates the direction of movement of the car.

  • Date: May 2015
